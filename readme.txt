The AdaptPose.ipynb file provides evaluation and training on 3DHP and 3DPW.  AdaptPose.ipynb can be run on Google Colab. All installations are provided and pre-trained models are also uploaded on an anonymous google drive account. AdaptPose.ipynb provides a step that downloads pre-trained models and the required data. 

Note: The implementation is based on the following work:

VideoPose3D: https://github.com/facebookresearch/VideoPose3D
PoseAuG: https://github.com/jfzhang95/PoseAug
